# Bogie Five
Robotic Programming for the Bogie Five Rover. The rover consists of the following parts:

* [Bogie Runt Rover Platform](https://www.servocity.com/bogie)
* [Raspberry Pi 3 B](https://www.raspberrypi.org/products/raspberry-pi-3-model-b/)
* [Raspberry Pi 8MP Camera](https://www.adafruit.com/products/3099)
* [Adafruit DC & Stepper Motor Hat](https://www.adafruit.com/product/2348)
* [Minibian: A minimal Raspian Operating System](https://minibianpi.wordpress.com/)
